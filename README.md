# .github

Organisation-wide community health files and profile for [Dharsi.ai](https://github.com/Dharsi-ai).

## Contents

| File / Folder | Purpose |
|---|---|
| `profile/README.md` | Organisation profile displayed on the [Dharsi-ai homepage](https://github.com/Dharsi-ai) |
| `SECURITY.md` | Vulnerability reporting policy (applies to all repos) |
| `CONTRIBUTING.md` | Contribution guidelines (applies to all repos) |
| `PULL_REQUEST_TEMPLATE.md` | Default pull request template (applies to all repos) |
| `ISSUE_TEMPLATE/` | Default issue templates — bug reports and feature requests |

These files are automatically inherited by all repositories in the organisation that do not define their own.
